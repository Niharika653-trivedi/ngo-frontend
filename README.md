🌐 Atmadeepam Society CRM (Frontend)

This is the frontend of the **Atmadeepam Society CRM Dashboard**, a web application designed to help NGOs efficiently manage students, donors, events, and organizational data.

 🚀 Live Demo

👉 https://ngo-frontend-olive.vercel.app/

 📌 Features

* 🔐 User Authentication (Login / Signup)
* 📊 Dashboard with analytics
* 👨‍🎓 Student Management
* 💰 Donor Management
* 📅 Event Management
* 📱 Responsive UI (works on mobile & desktop)

 🛠️ Tech Stack

* React.js (Vite)
* Tailwind CSS
* JavaScript (ES6+)
* REST API integration

 🔗 Backend API

This frontend is connected to a backend hosted on Render.
